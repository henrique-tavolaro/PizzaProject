# PizzaProject

This is the client side of a delivery app for a pizza place. I use in this project:
- Kotlin
- MVVM
- Dagger Hilt
- Google Sign In
- Firebase
- Room database
- Jetpack Compose
- Compose navigation
- Unit Tests
